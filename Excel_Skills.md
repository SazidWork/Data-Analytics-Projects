# Excel & Data Analysis Skills
Maine Satish Dhawale (Learn More) se Advanced Excel seekha hai. Main niche di gayi technical skills mein expert hoon:

### 📊 Data Visualization & Dashboards
- **Pivot Tables:** Raw data ko summarize karna aur reports banana.
- **Charts:** Bar charts, Pie charts aur Trend lines ka use karke data ko visually dikhana.
- **Slicers:** Interactive dashboards banana taaki data filter karna aasaan ho.

### 🔍 Formulas & Functions
- **VLOOKUP & XLOOKUP:** Bade data set mein se specific information dhoondhna.
- **Logical Functions:** IF, AND, OR ka use karke automatic data processing.
- **Data Validation:** Data entry mein mistakes rokne ke liye drop-down lists banana.

### 🛠 Data Cleaning
- **Conditional Formatting:** Duplicate values dhoondhna aur top performers ko highlight karna.
- **Text to Columns:** Data ko sahi format mein laana.
